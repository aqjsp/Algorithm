# 二叉树

### 定义

二叉树是一种树形结构，其中每个节点最多有两个子节点，分别称为左子节点和右子节点。

```
class TreeNode {
public:
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};
```

### 特点

1. 每个节点最多有两个子节点，分别称为左子节点和右子节点。
2. 左子节点和右子节点可以为空。
3. 二叉树的每个节点都包含一个值。
4. 通过递归的方式，可以将二叉树定义为一个根节点，以及左子树和右子树，其中左子树和右子树也都是二叉树。
