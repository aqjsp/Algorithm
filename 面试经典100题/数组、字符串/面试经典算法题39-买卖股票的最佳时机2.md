# 面试经典算法题39-买卖股票的最佳时机2

LeetCode.122

### 问题描述

给你一个整数数组 `prices` ，其中 `prices[i]` 表示某支股票第 `i` 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候最多只能持有一股股票。你也可以先购买，然后在同一天出售。

返回 你能获得的最大利润。

**示例 1：**

```
输入：prices = [7,1,5,3,6,4]
输出：7
解释：在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
     随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6 - 3 = 3 。
     总利润为 4 + 3 = 7 。
```

**示例 2：**

```
输入：prices = [1,2,3,4,5]
输出：4
解释：在第 1 天（股票价格 = 1）的时候买入，在第 5 天 （股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
     总利润为 4 。
```

**示例 3：**

```
输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 交易无法获得正利润，所以不参与交易可以获得最大利润，最大利润为 0 。
```

### 思路

1. 初始化最大利润 `maxProfit` 为 0。
2. 遍历数组，对于每一天的股票价格，计算该天卖出股票可以获得的利润。
3. 如果利润大于当前的最大利润，则更新最大利润。
4. 最终返回最大利润。

### 参考代码

#### C++

```
#include <iostream>
#include <vector>

using namespace std;

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int maxProfit = 0; // 初始化最大利润为0

        for (int i = 1; i < prices.size(); ++i) {
            int profit = prices[i] - prices[i - 1]; // 计算当天卖出的利润
            if (profit > 0) { // 如果利润大于0，更新最大利润
                maxProfit += profit;
            }
        }

        return maxProfit; // 返回最大利润
    }
};

int main() {
    vector<int> prices = {7, 1, 5, 3, 6, 4}; // 输入股票价格数组
    Solution solution;
    int result = solution.maxProfit(prices); // 计算最大利润
    cout << "最大利润：" << result << endl; // 输出结果

    return 0;
}
```
